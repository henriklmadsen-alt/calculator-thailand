## Internal Linking Strategy

### **Hub Linking** (from blog posts to calculators)
- Each post contains 3–4 contextual internal links to relevant calculators
- Links are placed naturally within content (not forced)
- Anchor text includes target keyword or calculator name
- Each post drives users to 1 primary calculator + 2–3 supporting calculators

### **Reverse Linking** (from calculators back to blog posts)
- Add "Related Article" section to each calculator page footer
- Link text: "[Blog post title] — Learn how to use this calculator effectively"
- Priority: Link from high-traffic calculator pages to blog posts
- Example on Net Salary Calculator footer:
  - "Read our guide: เครื่องคำนวณเงินเดือน: คู่มือฉบับสมบูรณ์"

### **Category Hub Linking**
- Create `/blog/` hub page listing all blog categories
- Link prominent calculators from blog hub:
  - "Financial Planning Calculators" section links to Net Salary, Loan, APR
  - "Financial Basics" section links to Income Tax, Unit Converter
  - "Life Planning" section (future expansion)

### **Internal Linking Metrics** (Track)
- Links from blog to calculators: 3–4 per post
- Links from calculators back to blog: 1 per relevant calculator
- Anchor text keyword density: 40–60% keyword-rich, 40–60% branded/generic
- Click-through rate from blog to calculator (GA4 goal)
