## Phase 2 Optimization Strategy

### 1. Metadata Optimization (All 12 Keywords)

**Apply to each keyword page (10 keywords have live calculator pages):**

#### Title Pattern (52-65 characters)
```
ตัวอักษร: "คำนวณ[ชื่อเครื่องคำนวณ] [ปี/ประเภท] ฟรี | Kamnuanlek"
```

**Examples:**
- ❌ Current: "คำนวณภาษีเงินได้บุคคล | Kamnuanlek"
- ✅ New: "คำนวณภาษีเงินได้บุคคล 2569 ฟรี | Kamnuanlek"
- ❌ Current: "คำนวณกองทุนสำรองเลี้ยงชีพ | Kamnuanlek"
- ✅ New: "คำนวณกองทุนสำรองเลี้ยงชีพ 2569 ฟรี | Kamnuanlek"

**Benefits:**
- Adds intent signals (year = authority, "ฟรี" = search intent clarity)
- Improves CTR in SERP (free calculator benefit stated upfront)

#### Meta Description Pattern (140-160 characters)
```
"คำนวณ[keyword] ตาม[legal/authority] [ปี] ฟรี — [x1.5 คูณตัวหลัก] ใส่[input field] รู้[benefit] ทันที"
```

**Examples:**
- ❌ Current: "คำนวณภาษีเงินได้บุคคลฟรี"
- ✅ New: "คำนวณภาษีเงินได้บุคคล 2569 ตามกฎหมายไทย ฟรี — ใส่เงินเดือน รู้ภาษีที่ต้องเสียทันที"
- ❌ Current: "เครื่องคำนวณกองทุน"
- ✅ New: "คำนวณกองทุนสำรองเลี้ยงชีพ 2569 ฟรี — ใส่เงินเดือนและสัญญา รู้เงินฝากประจำเดือนทันที"

**Benefits:**
- Matches Thai search intent (specific, actionable, benefit-driven)
- Improves CTR vs. generic descriptions
- Sets user expectations (free, instant results, specific inputs)

#### H1 Strengthening
- Current: Generic "คำนวณ[keyword]"
- New: "คำนวณ[keyword] 2569 — ตาม[legal authority]" + brief benefit line
